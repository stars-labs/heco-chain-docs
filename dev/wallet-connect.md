# 概述

Wallet connect 是使得移动 App 钱包通过扫描网页二维码或者Deeplink 的方式绑定账户到 DApp 的协议。

在使用这个协议的时候，DApp 与 移动 App 核心需要做的修改，是需要配置支持的 Heco 网络。
